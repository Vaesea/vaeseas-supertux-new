# Vaesea's SuperTux
A mod of SuperTux v0.7 that's inspired by the Forest World Adventure Pack and Tux Junior (it even (is planned to) has tux junior as the main character because he's cool)

This is a fork of PepperTux Legacy, however, that stuff will be removed from here. I decided to make a fork of it for this project because it's the most updated "SuperTux 0.5.1 in SuperTux 0.7" project I have, although it has different stuff like a planned Wallstone level so there's tiles for that in there which need to be removed from here. It also has different levels. It's a strange project.
